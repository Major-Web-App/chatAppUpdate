To connect the app to your firebase account first create a new project and enable the authorization of email and password and use the firestore database now create a web app and copy the four generated keys and now create a .env file and paste them in the below format

API_KEY=
AUTH_DOMAIN= 
PROJECT_ID= 
STORAGE_BUCKET= 
MESSAGING_SENDER_ID= 
APP_ID=

now install the yarn dependency and other dependencies used in project and use a emulator or use expo command line interface and install expo go app to run the app in android or ios
